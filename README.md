# PaperS3 User Demo

User demo source code of [PaperS3](https://docs.m5stack.com/en/core/PaperS3).

## Build

### Fetch Dependencies

```bash
python ./fetch_repos.py
```

### Tool Chains

[ESP-IDF v5.3.3](https://docs.espressif.com/projects/esp-idf/en/v5.3.3/esp32s3/index.html)

### Build

```bash
idf.py build
```

### Flash

```bash
idf.py flash
```
